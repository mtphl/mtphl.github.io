V580にCM14.1をインストール

 Sep 9, 2017
はじめに ZTE Blade V580というAndroid端末のCustom Rom(CM14.1)をがんばって入れた話し。のーわらんてぃ。 Links Stock Rom LineageOS TWRP IMEI修正プログラム SuperSu LineageOSのsu TT175=V580である。OneTeamとかいうフォーラムでダウンロードしたいときは、適当にアカウント作って親指をクリックすればできるようになる。 SP Flash Tool MediatekのSoCを使っている端末で、Preloaderをふさいでいないものは、SP Flash Toolというプログラムを使うことで、任意のパーティションを上書きできるようだ。 使い方はDownload押したあとに、電源を切った状態の端末に、USBケーブルをぶっさすだけ。nvramはバックアップしておいた方がよいかも。モデム領域もリセットしたいときは、Format All + Dowloadとかをつかうべし。 IMEIがぶっこわれたら SP Flash Toolを使っていると、端末固有の領域をフォーマットしなければならない状況に落ちいるので、あらかじめIMEIを控えておいてIMEI修正プログラムのimei.cのこの char out_mask[12] = {0xAB, 0xA0, 0x6F, 0x2F, 0x1F, 0x1E, 0x9A, 0x45, 0x0, 0x0, 0x0, 0x0}; 部分を char out_mask[12] = {0x16, 0xE5, 0xF4, 0xD1, 0xE8, 0x6A, 0x42, 0x40, 0x0, 0x0, 0x0, 0x0}; と書きかえてコンパイルしたブツに、引き数として控えておいたIMEIを与えてやる。(MT6753ではマスクデータだけ変わってるだけみたい)するとMP0B_001_NEWとかいうファイルができるのでそいつをadbとかで、/data/nvram/md/NVRAM/NVD_IMEI/MP0B_001に上書きしてやる。(要root)ちゃんとBaseband versionとかがみえている状況なら、IMEIが復活するハズ。(みえていない状況でもこの操作でいろいろ直ることはある) 1.BootloaderのUnlock まず、TWRPを焼いていろいろしたいので、BootloaderをUnlockしやす。この機種は設定からOEMロック解除とかいう項目にチェック入れた後に、fastboot modeでfastboot oem unlockとしてやればよい。その後、端末がリセットされるので、一回再起動しておこう。

Hugoの設定

 May 5, 2016
やりたいこと Mathjaxを使っていい感じの数式を書きたいのだが、_(アンダースコア)がMarkdownで予約されているせいで、Latexの下付き文字と衝突して困る。 いちいちエスケープするのも、そのままtexにもっていけないのでいやだ。 しょうがないので、Hugoのソースコードをいじってやった。 Goのインストール Goの最新バージョンをインストールして、pathの設定もしておく。 Hugoのダウンロード 公式ページにあるように、 go get -u -v github.com/gohugoio/hugo しておく。 Blackfridayにpatch 調べたところHugoはMarkdownからhtmlへ変換するために、Blackfridayなるパッケージを使っているらしい。ちょうど、やりたかったことをやってくださった方がいたので、その成果を拝借した。 patchを”$GOPATH/src/github.com/russross/blackfriday”に”git apply”する。 mirror <- こっちが最新(2017/9/17) その他のパッケージもつじつまをあわせる。 patchを”$GOPATH/src/github.com/cpuguy83/go-md2man”に”git apply”する。 patchを”$GOPATH/src/github.com/spf13/hugo”に”git apply”する。 build cd $GOPATH/src/github.com/gohugoio/hugo go build -o hugo main.go もしくは cd $GOPATH/src/github.com/gohugoio/hugo go install siteの設定 config.tomlに以下の設定を追加する。 extensions = ["latexMath"] 問題点 現在のバージョンではなぜか”hugo server”がうまく働かない。 現在(2016/6/18)は動くようだ。 また、バージョンがあがればこんなことをしなくても、config.tomlに extensions = ["noIntraEmphasis"] を追加すれば動く…かもしれない。 Blackfridayでは対応しない線が濃厚です。 Commands go get -u -v github.com/gohugoio/hugo cd $GOPATH/src/github.com/russross/blackfriday wget http://mtphl.com/hugo_settings/blackfriday.patch git apply blackfriday.

最近考えていること

 Jan 1, 0001
失敗に対する考え方の変化/自分が出来無いことについての自己正当化 すこし前までは, うまくいかないことがあると, 自分の努力が足りないからだと考えていた. しかし, 最近考えかたが変わってきた. どのように変化したか？ 以下の様な事を考えて, 自己正当化のような心理が働くようになったのだ. やればできるが, 人並みの水準に持っていくのにかかるコストが高い. そもそも, 自分のやりたい事ではなかった. やりたいこと(やるべきこと)だとしても, 周囲の期待する成功と自分の目指しているものがそもそもずれている. 失敗の指摘に対する反応も変化したように思う. 以前は叱られると萎縮してしまい, 自分はなんてダメなんだろうと考えていた. また, 周囲との軋轢を生まないような選択を無意識にしていた気がする. 近ごろは一歩引いて考えるようになり, 相手が不機嫌になったときでも 不機嫌になることでどのようなメッセージを与えたいのか, さらにいえば 自分をどうコントロールしたいのかという気配を探るようになった. ある種の開きなおりで, あまりよろしくないかと思ったけれども, ぼくは自分の気持ちを発見しなきゃいけないから, うけいれることにした. 視野を広くもちつつ, 自分の感情を大切にすることはどうすれば両立できるのだろうか？
